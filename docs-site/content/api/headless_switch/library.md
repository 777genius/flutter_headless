# headless_switch

Headless switch components.

Exports both the standalone switch and the list-tile variant. The components own interaction and state flow, while the active theme provides the visual renderer.

## Classes {#section-classes}

| Class | Description |
|---|---|
| [RSwitch](/api/src_presentation_r_switch/RSwitch) | Headless switch component for toggling a boolean value. |
| [RSwitchListTile](/api/src_presentation_r_switch_list_tile/RSwitchListTile) | Switch + label layout component for preference-style rows. |
| [RSwitchListTileStyle](/api/src_presentation_r_switch_list_tile_style/RSwitchListTileStyle) | Simple, Flutter-like styling sugar for [RSwitchListTile](/api/src_presentation_r_switch_list_tile/RSwitchListTile). |
| [RSwitchStyle](/api/src_presentation_r_switch_style/RSwitchStyle) | Simple, Flutter-like styling sugar for [RSwitch](/api/src_presentation_r_switch/RSwitch). |
