# slots

Slot override mechanisms for Headless components.

## Classes {#section-classes}

| Class | Description |
|---|---|
| [Decorate\<C\>](/api/src_slots_slot_override/Decorate) | Decoration wrapper around the default — receives child and can wrap/modify. |
| [Enhance\<C\>](/api/src_slots_slot_override/Enhance) | Enhancement of the default — transforms context before calling defaults. |
| [Replace\<C\>](/api/src_slots_slot_override/Replace) | Full replacement of a slot — default is ignored (intentional). |
| [SlotOverride\<C\>](/api/src_slots_slot_override/SlotOverride) | Base class for slot overrides in renderer contracts. |
