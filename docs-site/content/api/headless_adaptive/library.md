# headless_adaptive

Adaptive helpers for Headless.

Use this package when the app should choose platform-appropriate Headless presets automatically instead of wiring Material or Cupertino manually.

## Classes {#section-classes}

| Class | Description |
|---|---|
| [HeadlessAdaptiveApp](/api/headless_adaptive_app/HeadlessAdaptiveApp) | App bootstrap that selects the appropriate Headless preset for the current platform. |
| [HeadlessAdaptiveTheme](/api/src_headless_adaptive_theme/HeadlessAdaptiveTheme) | Theme provider that resolves adaptive Headless capabilities for a subtree. |
