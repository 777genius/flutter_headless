# headless

Facade package for the Headless ecosystem.

`headless` re-exports the core runtime, theme system, presets, and common
components so applications can depend on a single package for most use cases.

Included modules cover:

- tokens, foundation, contracts, and theme runtime
- Material and Cupertino presets
- button, checkbox, switch, dropdown, text field, and autocomplete components

Use this package when you want the standard Headless setup without managing
individual package imports yourself.
