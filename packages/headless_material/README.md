# headless_material

Material 3 preset for Headless components.

This package provides Material renderers and token resolvers for the Headless
runtime. It is intended to be used with the Headless component packages or via
the top-level `headless` facade.

Typical usage:

- `MaterialHeadlessTheme()` for theme installation
- `HeadlessMaterialApp(...)` for app bootstrap

Use this preset when you want Headless behavior with Material 3 visuals.
