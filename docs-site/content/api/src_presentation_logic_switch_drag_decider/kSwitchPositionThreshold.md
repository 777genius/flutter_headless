# kSwitchPositionThreshold

<div class="member-signature"><div class="member-signature-code"><span class="kw">const</span> <span class="type">double</span> <span class="fn">kSwitchPositionThreshold</span></div></div>

Position threshold for toggle decision when no fling detected.

If dragT >= 0.5, the switch toggles to "on".
If dragT < 0.5, the switch toggles to "off".
