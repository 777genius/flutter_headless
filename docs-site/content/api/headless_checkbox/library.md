# headless_checkbox

Headless checkbox components.

Exports both the standalone checkbox and the list-tile variant. Behavior stays in the component, while visuals come from the active theme renderer capabilities.

## Classes {#section-classes}

| Class | Description |
|---|---|
| [RCheckbox](/api/src_presentation_r_checkbox/RCheckbox) | Headless checkbox component for a single boolean value. |
| [RCheckboxListTile](/api/src_presentation_r_checkbox_list_tile/RCheckboxListTile) | Checkbox + label layout component with the same behavior model as [RCheckbox](/api/src_presentation_r_checkbox/RCheckbox). |
| [RCheckboxListTileStyle](/api/src_presentation_r_checkbox_list_tile_style/RCheckboxListTileStyle) | Simple, Flutter-like styling sugar for [RCheckboxListTile](/api/src_presentation_r_checkbox_list_tile/RCheckboxListTile). |
| [RCheckboxStyle](/api/src_presentation_r_checkbox_style/RCheckboxStyle) | Simple, Flutter-like styling sugar for [RCheckbox](/api/src_presentation_r_checkbox/RCheckbox). |
