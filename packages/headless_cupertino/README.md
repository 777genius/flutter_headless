# headless_cupertino

Cupertino preset for Headless components.

This package provides iOS-style renderers and token resolvers for the Headless
runtime. It works with the component packages directly or through the
top-level `headless` facade.

Typical usage:

- `CupertinoHeadlessTheme()` for theme installation
- `HeadlessCupertinoApp(...)` for app bootstrap

Use this preset when you want Headless behavior with Cupertino visuals.
