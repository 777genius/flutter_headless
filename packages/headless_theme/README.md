# headless_theme

Capability-based theme runtime for Headless.

`headless_theme` provides the theme container and override system used to
resolve renderers and tokens at runtime.

Core pieces include:

- `HeadlessTheme`
- `HeadlessThemeProvider`
- `RenderOverrides`
- debug helpers for consumed capabilities and overrides

Use this package to install a custom Headless theme or to scope visual
overrides to a subtree.
